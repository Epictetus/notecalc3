NoteCalc is a handy notepad with a smart builtin calculator.

Honestly, it just tries to be a free Soulver alternative in your browser.

Features

Simple and intuitive

NoteCalc is great as a simple calculator replacement. But it really shines when you use it to explore different scenarios and experiment with numbers. You can use words and numbers alongside each other and you almost never encounter an error. And all your work is automatically saved for you so you can reference it later.

Sum

Values are automatically accumulate in the sum variable.

Line references

Use the cursor keys while ALT key is pressed, to select and insert any previous value.
As you can see, if the cursor is on a line which contains references, both the reference and the referenced line are highlighted with the same color.





Variable/Reference usage highlighting

Variables and references are automatically highlighted at their usages, helping the reader.
Standing on a reference/variable and pressing CTRL + b jumps the cursor to the definition line of the reference/variable.

Autocompletion

Variable names are autocompleted if you type some of their starting letters and press TAB.
Below, "mo" was typed then a TAB was pressed, and there was only one variable starting with the letters "mo", so the app could unmistakably determine and autocomplete the variable.
If there is any name collision (e.g. a second variable would exist with name "mow", please specify more letters until the autocompletion is possible.
Coming soon Autocompletion popup dialog

Vectors and Matrices

You can create a matrix by typing a single m then press TAB.
While you are in matrix edit mode, you can resize the matrix with ALT and the cursor keys (e.g. ALT + up would decrease the number of rows).
You can finish matrix editing by navigating outside of the matrix with the cursor keys, or press Enter.





Binary operations

Bit manipulations (and, or, not, xor, shifts etc) are supported.

Binary and Hex Results

Pressing ALT + left/right changes the base of the current line's result to binary, hex and decimal.
Coming soon Scientific notation

Builtin functions

sin(), cos(), pi(), nth() and many more builtin functions
Coming soon Documentation





Unit conversion

Units, calculations with- and conversions between them are fully supported.

Easy sharing

The content is automatically saved into the URL, so just copy the URL and send it to a friend, she will see the same notes.

Automatic offline saving

The content is automatically saved into your offline browser storage, so if you leave the website and come back later, all your notes and tabs will be saved.
Your data is safe, nothing is sent to the server, everything is offline.
